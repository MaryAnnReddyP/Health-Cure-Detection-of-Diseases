# 🩺 HealthCure - Disease Detection System

## 📌 Overview

HealthCure is an AI-based web application that detects diseases using machine learning and deep learning models. It allows users to input symptoms or upload medical images to get real-time predictions.

## 🧠 Diseases Covered

* Brain Tumor
* Breast Cancer
* Alzheimer’s
* Diabetes
* Pneumonia
* Heart Disease

## ⚙️ Technologies

* Python
* Flask
* Machine Learning
* TensorFlow
* HTML, CSS

## 🚀 How to Run

1. Install requirements
2. Run app.py
3. Open localhost
